# Cross-Selling Opportunities Detection with Machine Learning

## Overview

The goal of this project is to detect associations between product categories and translate them into actionable cross-selling strategies to increase average ticket and revenue.

The solution integrates:
- 📊 Exploratory Data Analysis (EDA)
- 🤖 Machine Learning model (Random Forest)
- 🖥️ Interactive app (Streamlit)
- 📈 Business dashboard (Power BI)

---

## Key Insights

- Strong association detected between **Home** and **Recreation** categories
- Cross-selling potential identified with:
  - 📈 +19.4% estimated revenue increase
  - 🎯 100% activation potential
- Clear prioritization of opportunities (High / Medium / Low)

---

## Model

- Algorithm: **Random Forest**
- Objective: Predict cross-selling opportunities
- Evaluation metrics:
  - Accuracy: 0.66
  - Recall: 1.00
  - F1 Score: 0.73

The model prioritizes recall to ensure all potential opportunities are captured.
